# 有限会社アスアアーツ コーポレートサイト

文化とエンターテインメントの力で人と社会を豊かにする、有限会社アスアアーツの公式Webサイトです。

## サイト構成

- **トップページ** - ヒーロースライダー（四季の画像5枚）
- **会社概要** - 企業理念・事業紹介
- **事業内容** - 人材育成・派遣、地域活性化、企業マッチング
- **実績紹介** - これまでの活動実績
- **代表紹介** - 代表取締役のプロフィール
- **会社情報** - 基本情報
- **お問い合わせ** - コンタクトフォーム
- **プライバシーポリシー** - 個人情報保護方針

## 技術スタック

- HTML5
- CSS3（カスタムプロパティ、Flexbox、Grid、レスポンシブデザイン）
- JavaScript（Vanilla JS）
- Google Fonts（Shippori Mincho、Cormorant Garamond）

## ファイル構成

```
├── index.html          # メインページ
├── privacy.html        # プライバシーポリシー
├── css/
│   └── style.css       # スタイルシート
├── js/
│   └── main.js         # JavaScript
└── images/
    ├── logo.png        # ロゴ
    ├── hero0.png       # ヒーロー画像（四季まとめ）
    ├── hero1.png       # ヒーロー画像（春）
    ├── hero2.png       # ヒーロー画像（夏）
    ├── hero3.png       # ヒーロー画像（秋）
    ├── hero4.png       # ヒーロー画像（冬）
    ├── about.png       # 会社概要画像
    └── representative.png  # 代表紹介画像
```

## レスポンシブ対応

| 画面幅 | 対象デバイス |
|--------|-------------|

| 480px以下 | スマホ |
