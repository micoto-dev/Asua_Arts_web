# 有限会社アスアアーツ コーポレートサイト

文化とエンターテインメントの力で人と社会を豊かにする、有限会社アスアアーツの公式Webサイトです。

## サイト構成

- **トップページ** - ヒーロースライダー（四季の画像）
- **会社概要** - 企業理念・事業内容
- **事業内容** - 人材育成・派遣、地域活性化、企業マッチング
- **実績紹介** - これまでの活動実績
- **代表紹介** - 代表取締役のプロフィール
- **会社情報** - 基本情報
- **お問い合わせ** - コンタクトフォーム

## 技術スタック

- HTML5
- CSS3（カスタムプロパティ、Flexbox、Grid）
- JavaScript（Vanilla JS）
- Google Fonts（Shippori Mincho、Cormorant Garamond）
